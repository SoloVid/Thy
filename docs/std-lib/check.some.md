```thy
a is given Boolean
b is given Boolean true

type return Boolean
```

Check if some of the supplied terms are true 
([logical OR operation](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Logical_OR)).

**Example:**

```thy
a is getCondition1
b is getCondition2
c is getCondition3
check.some a b c
if that
  print "a, b, or c true"
```
